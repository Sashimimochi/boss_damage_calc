# Boss Damage Calculator

## これは何？
リアルタイムに画面をキャプチャして、与ダメージを計算するためのツール

最初にマウスポインタで対象領域を指定して、その範囲の文字認識を行う

## 準備
### 外部ライブラリ
- Tesseract OCR
  - https://github.com/UB-Mannheim/tesseract/wiki

### Pythonライブラリ
```Console
$ pip install -r requirements.txt
```

## 実行方法
```Console
$ python app.py
```

### Option
- `export log_level=debug`: デバッグメッセージを表示する

## 使い方
基本的に、コンソールのメッセージに従う。

1. 実行するとキャプチャ範囲を指定するためのウィンドウが表示されるので、ドラッグして範囲を決める
1. キャプチャ範囲を決定したらウィンドウを閉じる
1. 敵の初期HPをコンソールから入力する
1. 一定の間隔ごとに選択した範囲のスクリーンショットが撮られ、HP計算処理結果がコンソールに表示される
1. 計算結果に間違いがあるときは手動で修正する
1. コンソールで`Ctrl+C`を押すと終了できる

`output/hp.txt`に現在のHPが出力される。
直接修正値を入力してもよいし、差分を入力してもよい。

example 1
```txt
9000
```
example 2
```txt
8500+500
```
